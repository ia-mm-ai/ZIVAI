# ZIVAI POSTURE

ZIVAI is a read-only surface that emits capacity state.

ZIVAI does:
- emit a single capacity state
- expose the signal publicly
- remain non-executing

ZIVAI does not:
- act
- decide
- enforce
- infer intent
- store identity
- persist behavior across events
- trigger workflows
- claim authority

Output semantics:
- informational only
- any downstream use is external and out-of-scope
